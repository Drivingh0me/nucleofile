# How to use nucleophile
The two ways to interact with nucleophile are via the args when calling
the program and with a control file.

## Flags
There are multiple flags for nucleophile. This is a comprehensive list
in the format: __-short__/__--long__ Description, arg description.
(number of args)

- __-o__/__--open__ Open a file, file path. (1)
- __-s__/__--silent__ Execute silently.

## How to use a control file
A control file __MUST__ be .bat or .ctrl files.

## Saving a file
The filetype for nucleophile sessions is .nucf. This is not a standard
format, but a specification is defined. These .nucf files are txt based
and can easily be parsed.
